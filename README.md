# My dotfiles

## Components

- NVIM (.config/nvim)
- EMACS (.emacs.d)

## How to configure

1. Clone this Repository to any local directory
2. Create a Symlink to your actual home directory

### Appendix Symlink

```bash
ln -s $HOME/Documents/DevProjects/dotfiles/.emacs.d $HOME/.emacs.d
ln -s $HOME/Documents/DevProjects/dotfiles/.config/nvim $HOME/.config/nvim
```
